Key Features

Advanced safety technology for GPS-denied environments

IMU-Based Detection

Real-time motion monitoring at 50Hz using 6-DOF accelerometer and gyroscope

Multi-Condition Analysis

Detects impact, rollover, and spin with 2-of-3 confirmation to reduce false positives

RF Beacon System

433MHz emergency broadcast with 500m+ range, no GPS or cellular required

RSSI Homing

Signal strength-based localization for rescue teams to find accident site

Fully Autonomous

Zero driver interaction required, automatic activation on accident detection

Cost-Effective

Complete system for ₹3,694, scalable for fleet deployment

System Architecture

Two-unit design for comprehensive coverage

Vehicle Unit

Components

  • ESP32 Microcontroller
  • MPU-6050 IMU Sensor
  • HC-12 RF Module (433MHz)
  • 12V to 5V Buck Converter
  • Status LEDs & Override Button

Functions

  • Continuous motion monitoring
  • Accident detection algorithm
  • RF beacon transmission
  • Manual override capability
Cost: ₹1,515

Receiver Unit

Components

  • ESP32 Microcontroller
  • HC-12 RF Module (433MHz)
  • 16x2 LCD Display (I2C)
  • RSSI Bar (5 LEDs)
  • Buzzer & USB Power Bank

Functions

  • Emergency signal reception
  • Vehicle ID display
  • RSSI homing guidance
  • Audio/visual alerts
Cost: ₹2,179
